# twenty-four
A python script to break St FX's Math Club game

## Description
Given numbers a, b, c, d and a target t, generates a set of operations where `a [op1] b [op2] c [op3] d = t`. Note that there will be some "duplicates".


Ex.

If a, b, c, d = 4 and t = 24, one such possible operation set is:

`(4)+(4)+(4)*(4)`

## Usage

Use the function, `generate([a, b, c, d], t)`
